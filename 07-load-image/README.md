### Загрузка и обработка изображений

1. Установите зависимости
```npm i```

2. Запустите сервер
Выполните из корневой директории репозитория:
```node 06-load-image```

3. Откройте в браузере
```http://localhost:3000```
Выберите изображение (JPG, PNG, GIF или WebP), нажмите «Загрузить» — и вы увидите обработанную версию!

### Цель проекта
Этот пример демонстрирует реальный сценарий backend-обработки файлов, который используется в блогах, соцсетях, CMS и других веб-приложениях.
